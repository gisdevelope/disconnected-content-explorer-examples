DICE
Content Creation Guide

Offline content doesn't have to be just another PDF or Powerpoint presentation.

With DICE you can have rich, interactive HTML with video, animations, and slippy maps. Don't worry, your PDFs and Microsoft Office documents can come too.

Creating content for DICE

Reports in DICE are displayed in a web view. While you can load PDFs and Office documents, interactive web content is much more compelling. Why just show a static map when you can allow the user to pan around and zoom? When creating websites to view in DICE you can use HTML, JavaScript, and CSS. Since you cannot install browser plugins in mobile browsers; Flash, Silverlight, or applets will not work in DICE.

Just as with normal websites, the root folder of your report must contain an index.html file. The rest of your resources and can be placed in folders. Relative links to files are supported by DICE. The image to the right shows the layout for the COFlood demo report which uses several JavaScript libraries, displays a map, and also has a video embedded in the page.

Loading reports onto your iPad

DICE iOS supports PDFs, Microsoft Office Documents and websites contained in .zip files. To load them into the app, connect the iPad to your computer and open up iTunes. Click on the device name on the tool bar, and then follow these steps below.

To delete reports, select items from the DICE Documents list and press the delete key, then click the sync button.

Once the data has been synced to your iPad, open DICE, the first time it may take a few seconds to load. While unzipping the content the report description in the list view will update with the unzip progress. If the app is already open when you sync, pull down on the report list to refresh. A spinner will display showing that the reports are being loaded.

Tap on a report to view it.

PDF and Office files can be loaded into the app without being zipped.

Loading reports onto your Android tablet

If you are using a Mac you may need to install Android File Transfer, available at: https://www.android.com/filetransfer/. To load reports on the Android version, connect your device to your computer, on the SD Card create a DICE folder. Drag and drop the report zip files into the DICE folder. The app will detect that content has been added and will load it.

The Android and iOS versions can view the same report zip files. You can load PDF and office docs into DICE on Android, however viewing those types of content may require you to install additional apps to display them properly.

Recommended Frameworks

Feel free to pull this report out of DICE and poke around in the code.

  • Twitter Bootstrap for layout - http://getbootstrap.com/
  • TwentyTwenty used for the image compare at the top of this page - http://zurb.com/playground/twentytwenty
  • LeafletJS for maps - http://leafletjs.com/
  • JQuery for manipulating elements on the page - http://jquery.com/
  • AngularJS for more complex interactions that require a data model - http://angularjs.org/
  • D3 for really fancy charts - http://d3js.org/
  • Swiper for touchable photo galleries - http://www.idangero.us/sliders/swiper/index.php
  • HammerJS for multitouch interactions - http://eightmedia.github.io/hammer.js/
  • One Page Scroll for layouts similar to the Circa app - https://github.com/peachananr/onepage-scroll
